# RV32I 命令セット

## オペコード一覧

| opcode[6:0] | 名称 | 命令グループ |
|:-----------|------|------------|
| `0110011` | OP | レジスタ–レジスタ演算 / M 乗算除算 |
| `0010011` | OP-IMM | 即値演算 |
| `0000011` | LOAD | ロード |
| `0100011` | STORE | ストア |
| `1101111` | JAL | ジャンプ・アンド・リンク |
| `1100111` | JALR | ジャンプ・アンド・リンク・レジスタ |
| `1100011` | BRANCH | 条件分岐 |
| `0110111` | LUI | 上位即値ロード |
| `0010111` | AUIPC | PC + 上位即値 |
| `0001111` | MISC-MEM | フェンス |
| `1110011` | SYSTEM | システム / CSR |
| `0101111` | AMO | アトミック操作 (A) |
| `0000111` | LOAD-FP | FP ロード (F) |
| `0100111` | STORE-FP | FP ストア (F) |
| `1010011` | OP-FP | FP 演算 (F) |

## ロード命令 (I形式) — opcode = `0000011`

| 命令 | funct3 | 説明 |
|------|:------:|------|

| LW   | `010`  | 語をロード |
| LBU  | `100`  | 符号無しバイトをロード |
| LHU  | `101`  | 符号無し半語をロード |

## 即値演算命令 (I形式) — opcode = `0010011`

| 命令 | funct3 | funct7 / 備考 | 説明 |
|------|:------:|--------------|------|
| ADDI | `000`  | — | 即値加算 |
| SLTI | `010`  | — | 符号付き即値比較 (rs1 < imm) |
| SLTIU| `011`  | — | 符号無し即値比較 (rs1 < imm) |
| XORI | `100`  | — | 即値排他的論理和 |
| ORI  | `110`  | — | 即値論理和 |
| ANDI | `111`  | — | 即値論理積 |
| SLLI | `001`  | funct7=`0000000`, imm[11:5]=`00000` | 即値論理左シフト (shamt[4:0]) |

## RV32A — アトミック拡張

全命令 opcode = `0101111`, funct3 = `010`, 形式は R 形式に aq/rl を追加。

```
31:27    26:25   24:20   19:15   14:12   11:7    6:0
funct5  | aq | rl | rs2  | rs1   | funct3 | rd    | opcode
```

### LR/SC

| 命令 | funct5 | aq | rl | 説明 |
|------|:------:|:--:|:--:|------|
| LR.W  | `00010` | 任意 | 任意 | 予約付きロード (rd = Mem[rs1], 予約セット) |
| SC.W  | `00011` | 任意 | 任意 | 条件付きストア (成功: rd=0, 失敗: rd!=0) |

### AMO (Atomic Memory Operation)

| 命令 | funct5 | 説明 |
|------|:------:|------|
| AMOSWAP.W | `00001` | アトミック交換 (rd = Mem[rs1], Mem[rs1] = rs2) |
| AMOADD.W  | `00000` | アトミック加算 |
| AMOXOR.W  | `00100` | アトミック XOR |
| AMOAND.W  | `01100` | アトミック AND |
| AMOOR.W   | `01000` | アトミック OR |
| AMOMIN.W  | `10000` | アトミック符号付き MIN |
| AMOMAX.W  | `10100` | アトミック符号付き MAX |
| AMOMINU.W | `11000` | アトミック符号無し MIN |
| AMOMAXU.W | `11100` | アトミック符号無し MAX |

## RV32F — 単精度浮動小数点拡張

### FP ロード・ストア

| 命令 | 形式 | opcode | funct3 | 説明 |
|------|:----:|:------:|:------:|------|
| FLW | I | `0000111` | `010` | 単精度浮動小数点ロード (rd = float) |
| FSW | S | `0100111` | `010` | 単精度浮動小数点ストア (rs2 = float) |

### FP 演算 (R形式) — opcode = `1010011`

`funct7 | rs2 | rs1 | rm | rd | 1010011`

| 命令 | funct7 | rm(※) | 説明 |
|------|:------:|:-----:|------|
| FADD.S    | `0000000` | rm | 加算 |
| FSUB.S    | `0000100` | rm | 減算 |
| FMUL.S    | `0001000` | rm | 乗算 |
| FDIV.S    | `0001100` | rm | 除算 |
| FSQRT.S   | `0101100` | rm | 平方根 (rs2=`00000`) |
| FSGNJ.S   | `0010000` | `000` | 符号付加 (fsgnj) |
| FSGNJN.S  | `0010000` | `001` | 符号否定付加 (fsgnjn) |
| FSGNJX.S  | `0010000` | `010` | 符号排他的論理和付加 (fsgnjx) |
| FMIN.S    | `0010100` | `000` | 最小値 |
| FMAX.S    | `0010100` | `001` | 最大値 |

> ※ rm (rounding mode): bits 14:12 = `000`(RNE) `001`(RTZ) `010`(RDN) `011`(RUP) `100`(RMM) `111`(dyn). FSGNJ/FMIN/FMAX では funct3 として動作。

### FMA (R4形式) — opcode = `1010011`

```
31:27   26:25   24:20   19:15   14:12   11:7    6:0
rs3    | fmt=00 | rs2   | rs1   | rm     | rd    | opcode
```

| 命令 | rs3 | 説明 |
|------|:---:|------|
| FMADD.S   | 乗数 | $\text{rd} = \text{rs1} \times \text{rs2} + \text{rs3}$ |
| FMSUB.S   | 乗数 | $\text{rd} = \text{rs1} \times \text{rs2} - \text{rs3}$ |
| FNMSUB.S  | 乗数 | $\text{rd} = -(\text{rs1} \times \text{rs2}) + \text{rs3}$ |
| FNMADD.S  | 乗数 | $\text{rd} = -(\text{rs1} \times \text{rs2}) - \text{rs3}$ |

### FP 比較 — opcode = `1010011`, funct7 = `1010000`

| 命令 | funct3 | 説明 |
|------|:------:|------|
| FEQ.S | `010` | 等しい ($\text{rd} = 1$ if rs1 == rs2) |
| FLT.S | `001` | 小なり ($\text{rd} = 1$ if rs1 < rs2) |
| FLE.S | `000` | 以下 ($\text{rd} = 1$ if rs1 <= rs2) |

### FP 変換 — opcode = `1010011`

| 命令 | funct7 | funct3 | rs2 | 説明 |
|------|:------:|:------:|:---:|------|
| FCVT.W.S   | `1100000` | rm | `00000` | float → 符号付き整数 |
| FCVT.WU.S  | `1100000` | rm | `00001` | float → 符号無し整数 |
| FCVT.S.W   | `1101000` | rm | `00000` | 符号付き整数 → float |
| FCVT.S.WU  | `1101000` | rm | `00001` | 符号無し整数 → float |

### FP 移動・分類 — opcode = `1010011`

| 命令 | funct7 | funct3 | rs2 | 説明 |
|------|:------:|:------:|:---:|------|
| FMV.X.W    | `1110000` | `000` | `00000` | float ビット列を整数 rd へ移動 |
| FCLASS.S   | `1110000` | `001` | `00000` | float 分類 (rd に 10bit マスク) |
| FMV.W.X    | `1111000` | `000` | `00000` | 整数ビット列を float rd へ移動 |

### 丸めモード (rm)

| rm[2:0] | モード | 動作 |
|:-------:|--------|------|
| `000` | RNE | 最近接偶数丸め |
| `001` | RTZ | ゼロ方向切捨て |
| `010` | RDN | 負の無限大方向切捨て |
| `011` | RUP | 正の無限大方向切上げ |
| `100` | RMM | 最近接最大絶対値丸め |
| `111` | DYN | frm CSR から動的取得 |

## RV32C — 圧縮命令拡張 (16-bit)

全命令の上位 2bit (bits 1:0) で象限を判別:

| bits[1:0] | 象限 | 形式 |
|:---------:|------|:----:|
| `00` | Quadrant 0 | CIW, CL, CS |
| `01` | Quadrant 1 | CI, CJ, CB |
| `10` | Quadrant 2 | CI, CSS |
| `11` | — | 非圧縮命令 (32-bit) |

### Quadrant 0 (bits[1:0] = `00`)

形式: `funct3 | imm/rd' | imm/rs1' | imm/rs2' | 00`

| funct3 | 命令 | 形式 | RV32C | 説明 |
|:------:|------|:----:|:-----:|------|
| `000` | C.ADDI4SPN | CIW | ○ | rd' = sp + nzuimm (nzuimm≠0) |
| `001` | C.FLW | CL | ○(F) | rd' = float Mem[rs1'+uimm] |
| `010` | C.LW | CL | ○ | rd' = Mem[rs1'+uimm] |
| `100` | reserved | — | — | — |
| `101` | C.FSW | CS | ○(F) | Mem[rs1'+uimm] = rs2' float |
| `110` | C.SW | CS | ○ | Mem[rs1'+uimm] = rs2' |

> rd'/rs1'/rs2' は x8–x15 に対応 (3bit レジスタ指定 + 8)。

### Quadrant 1 (bits[1:0] = `01`)

形式: `funct3 | imm/rd | imm/rs1 | imm/rs2 | 01`

| funct3 | 命令 | 形式 | RV32C | 説明 |
|:------:|------|:----:|:-----:|------|
| `000` | C.ADDI | CI | ○ | rd = rd + nzimm; rd=x0 → C.NOP |
| `001` | C.JAL | CJ | ○(RV32) | x1 = PC+2, PC += offset |
| `010` | C.LI | CI | ○ | rd = imm (rd≠x0) |
| `011` | C.ADDI16SP | CI | ○ | sp = sp + imm (rd=x2) |
| `100` | 下記参照 | — | ○ | 演算系命令群 |
| `101` | C.J | CJ | ○ | PC += offset |
| `110` | C.BEQZ | CB | ○ | if (rs1' == 0) PC += offset |
| `111` | C.BNEZ | CB | ○ | if (rs1' != 0) PC += offset |

funct3=`100` の拡張 (bits[12:10] で分岐):

| bits[12:10] | 命令 | 形式 | 説明 |
|:----------:|------|:----:|------|
| `00` | C.SRLI | CI | rs1' = rs1' >> shamt (shamt[5]=0) |
| `01` | C.SRAI | CI | rs1' = rs1' >>> shamt (shamt[5]=0) |
| `10` | C.ANDI | CI | rs1' = rs1' & imm |
| `11` | 下記 | CA | bit[12]=1, bits[6:5] でさらに分岐 |

bits[12:10]=`11` の拡張:

| bits[6:5] | funct2[1:0] | 命令 | 説明 |
|:---------:|:-----------:|------|------|
| `00` | — | C.SUB | rs1' = rs1' - rs2' |
| `01` | — | C.XOR | rs1' = rs1' ^ rs2' |
| `10` | — | C.OR | rs1' = rs1' \| rs2' |
| `11` | `00` | C.AND | rs1' = rs1' & rs2' |
| `11` | `01` | C.SUBW | rs1' = rs1' - rs2' (RV64) |
| `11` | `10` | C.ADDW | rs1' = rs1' + rs2' (RV64) |
| `11` | `11` | reserved | — |

### Quadrant 2 (bits[1:0] = `10`)

形式: `funct3 | imm/rd | imm/rs1 | imm/rs2 | 10`

| funct3 | 命令 | 形式 | RV32C | 説明 |
|:------:|------|:----:|:-----:|------|
| `000` | C.SLLI | CI | ○ | rd = rd << shamt (shamt[5]=0) |
| `001` | C.FLWSP | CI | ○(F) | rd = float Mem[sp+uimm] (rd≠x0) |
| `010` | C.LWSP | CI | ○ | rd = Mem[sp+uimm] (rd≠x0) |
| `100` | 下記参照 | CR | ○ | レジスタ間操作 |
| `101` | C.FSWSP | CSS | ○(F) | Mem[sp+uimm] = rs2 float |
| `110` | C.SWSP | CSS | ○ | Mem[sp+uimm] = rs2 |

funct3=`100` の拡張:

| bit[12] | 条件 | 命令 | 説明 |
|:-------:|------|------|------|
| `0` | rs2=x0 | C.JR | PC = rs1 (rd=x0) |
| `0` | rs2≠x0 | C.MV | rd = rs2 |
| `1` | rs1=x0, rs2=x0 | C.EBREAK | ブレークポイント |
| `1` | rs1≠x0, rs2=x0 | C.JALR | x1 = PC+2, PC = rs1 |
| `1` | rs2≠x0 | C.ADD | rd = rs1 + rs2 |

### RV32C 命令形式

```
CIW:   15:13    12:5      4:2   1:0
      | funct3 | nzuimm  | rd'  | 00 |

CL:    15:13  12:10   9:7    6:5    4:2   1:0
      | funct3 | imm   | rs1' | imm  | rd'  | 00 |

CS:    15:13  12:10   9:7    6:5    4:2   1:0
      | funct3 | imm   | rs1' | imm  | rs2' | 00 |

CI:    15:13  12    11:7    6:2    1:0
      | funct3 | imm | rd/rs1| imm | 01 |

CJ:    15:13  12:11  10:7   6    5   4:1    0   1:0
      | funct3 | imm  | imm | imm|imm| imm | imm| 01 |

CB:    15:13  12:10  9:7   6:5   4:3   2:1    0   1:0
      | funct3 | imm  | rs1'| imm | imm | imm | imm| 01 |

CR:    15:12    11:7    6:2    1:0
      | funct4 | rd/rs1| rs2  | 10 |

CSS:   15:13   12:7    6:2    1:0
      | funct3 | imm   | rs2  | 10 |

CA:    15:13   12    11:10   9:7    6:5    4:2    1:0
      | funct3 | 1   | funct2| rs1' | funct | rs2' | 01 |
```

## 命令フォーマット (32-bit)

```
        31:25    24:20    19:15    14:12    11:7     6:0
R形式: | funct7  | rs2    | rs1    | funct3 | rd     | opcode  |
I形式: | imm[11:0]        | rs1    | funct3 | rd     | opcode  |
S形式: | imm[11:5]| rs2    | rs1    | funct3 | imm[4:0]| opcode |
B形式: | imm[12|10:5]| rs2 | rs1    | funct3 | imm[4:1|11]|opcode|
U形式: | imm[31:12]                | rd     | opcode  |
J形式: | imm[20|10:1|11|19:12]    | rd     | opcode  |
R4形式: | rs3   | fmt   | rs2    | rs1    | rm     | rd     | opcode |
```

### 即値エンコーディング

| 形式 | 即値構成 |
|:----:|---------|
| I | `imm[11:0]` (12bit 符号付き) |
| S | `imm[11:5]` → 31:25, `imm[4:0]` → 11:7 (12bit 符号付き) |
| B | `imm[12|10:5]` → 31:25, `imm[4:1|11]` → 11:7 (13bit 符号付き, LSB=0) |
| U | `imm[31:12]` → 31:12 (32bit, 下位12bit=0) |
| J | `imm[20|10:1|11|19:12]` → 31:12 (21bit 符号付き, LSB=0) |
